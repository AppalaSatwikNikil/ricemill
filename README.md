# Santi Rice Mill

A premium e-commerce frontend for Santi Rice Mill built with React + Vite.

## Getting Started

### Prerequisites
- Node.js installed.

### Installation

1. Navigate to the project directory:
   ```bash
   cd d:\Ricemill
   ```
2. Install dependencies (if not already installed):
   ```bash
   npm install
   ```

### Running the App

To start the development server:

```bash
npm run dev
```

This will start the server at `http://localhost:5173` and automatically open it in your default browser.

### Building for Production

```bash
npm run build
```
